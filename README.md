# LolehRote.com
Portal berita, budaya, dan pariwisata Pulau Rote (NTT) berbasis PHP + MySQL + Bootstrap 5.

## Fitur Utama
- Artikel & Kategori (premium support)
- Komentar, Like/Dislike
- Video (YouTube & lokal)
- Pencarian live
- RSS & Sitemap
- Admin panel lengkap
- Pembayaran premium (Midtrans)
- PWA-ready (bisa di-install)

## Persyaratan
- PHP 8.0+
- MySQL 5.7+
- Composer (untuk Midtrans)
- HTTPS (untuk PWA & payment)
